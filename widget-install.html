Vogo.Family Chat Widget Installation Guide

Option 1: Using Custom HTML Widget

  1. Go to Appearance > Widgets in your WordPress admin
  2. Add a "Custom HTML" widget to your desired widget area
  3. Paste the following code:
<!-- Vogo.Family Chat Widget -->
<div id="vogo-chat-root"></div>
<script src="https://chatbot-support-1-9ljn.onrender.com/static/js/main.cacfdeba.js"></script>
<link href="https://chatbot-support-1-9ljn.onrender.com/static/css/main.cda35c4f.css" rel="stylesheet">

Option 2: Using Theme Customizer (Footer Code)

  1. Go to Appearance > Customize
  2. Look for "Additional CSS" or "Custom HTML/JavaScript" section
  3. Paste the widget code in the footer section

Option 3: Using a Plugin

  1. Install the "Header and Footer Scripts" plugin
  2. Go to Settings > Header and Footer Scripts
  3. Add the widget code to the "Scripts in Footer" section

Option 4: Editing theme's footer.php

  1. Go to Appearance > Theme Editor
  2. Select your theme's footer.php file
  3. Add the widget code just before the closing </body> tag
  4. Click "Update File" to save
Note: If you update your theme, any changes made directly to theme files will be lost. Using a child theme or one of the other methods is recommended.
